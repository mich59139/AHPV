Guide d'utilisation

🗺️ Navigation

  • Cliquez sur les marqueurs pour voir les articles du lieu
  • Utilisez la molette ou +/- pour zoomer
  • Glissez pour déplacer la carte
  • Cliquez sur ⌂ pour recentrer

🔍 Filtres

  • Bouton ☰ : Ouvrir les filtres
  • Filtrez par thème, époque ou numéro
  • La recherche trouve titre, auteur, mots-clés

📊 Marqueurs

  • La taille indique le nombre d'articles
  • Les clusters regroupent les lieux proches
  • Cliquez sur un cluster pour zoomer
